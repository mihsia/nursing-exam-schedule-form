# 護理師甄選複試時間分配表產生器

開啟 `app/index.html` 即可使用複試時間分配表。
開啟 `app/grouping.html` 可使用成績 S 型分組工具。

## 使用方式

1. 填寫標題、日期、組別與第一位報到時間。
2. 貼上准考證號碼，一行一筆，或用逗號、空白分隔。
3. 設定各階段與報到時間的分鐘差：
   - 引導抽題
   - 實作抽題準備開始與長度
   - 實作開始與長度
   - 口試預備教室
   - 口試開始與長度
4. 如需休息或午休，在休息規則中逐行設定。

休息規則格式：

```text
第幾位後:休息分鐘:顯示文字
```

範例：

```text
7:15:休息
14:55:午休
21:15:休息
```

## 輸出

- `列印 / 另存 PDF`：使用瀏覽器列印功能，可選擇另存成 PDF。
- `匯出 CSV`：匯出可用 Excel 開啟的時間分配表資料。
- `載入附件範例`：恢復成附件 A 組的範例資料與時間設定。

## 成績 S 型分組

1. 開啟 `app/grouping.html`。
2. 匯入含有 `准考證` 與 `成績` 欄位的 `.xlsx`、`.csv` 或 `.tsv`。
3. 輸入組數，最多 26 組。
4. 點選 `產生 S 型分組`。
5. 使用 `匯出 CSV` 產出 `組別`、`准考證` 欄位資料。

分組方式：先依 `成績` 由高到低排序，再依 A、B、C... 下一排反向 C、B、A... 的 S 型方式分配。
